# Ai4Energy Docs - Development Tools & Workflow

Ai4Energy 文档 - 开发工具与工作流

## 内容

- **CS 进阶**: 抽象、递归、DSL 等计算机科学进阶概念
- **开发工具**: VSCode、Git、CoolProp 等开发工具使用
- **工作流**: Julia 环境配置、Git 工作流最佳实践
- **go-zero 教程**: go-zero 微服务框架完整教程

## 部署

本项目使用 VitePress 构建，部署在 GitHub Pages：
https://huxinyang-web.github.io/ai4edocs-dev-tools/

