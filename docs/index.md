# 开发工具与工作流

欢迎来到开发工具与工作流课程。本课程涵盖计算机科学进阶概念、开发工具链使用、Julia 工作流最佳实践，以及 go-zero 微服务框架完整教程。

## 课程内容

### CS 进阶
过程抽象、数据抽象、递归、符号到模型、抽象语法树、代码生成、领域特定语言、库与语言设计，以及 Ascend 和 OpenFoam 解构分析。

### 开发工具
VSCode 与 Git 协作、CoolProp 物性库、CSV.jl 数据处理、SSH 与 Git 实操。

### 工作流
Julia 安装与 REPL、包服务器切换、Git 工作流、学习思维、VSCode 插件、文档结构规范。

### go-zero 教程
Docker Compose 环境准备、EMQX 消息服务器、MySQL 数据库、go-zero 框架搭建、Gateway 配置、goctl 生成 API 服务和数据库模型。
